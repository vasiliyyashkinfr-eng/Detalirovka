# Деталировка 3D

Калькулятор деталировки корпусной мебели с 3D-визуализацией на Three.js.
Работает в браузере на десктопе и телефоне (адаптивный интерфейс + PWA, офлайн).

## Что умеет (MVP)

- **Генератор корпуса** — задаёшь габариты (Ш×В×Г), толщину ЛДСП, число полок,
  наличие крыши/задней стенки — детали раскидываются автоматически.
- **Ручная сборка** — добавляй и правь любые детали: размеры, материал, ориентация,
  кромка по 4 сторонам, направление текстуры, количество.
- **3D-сцена** (react-three-fiber): вращение/зум/панорама (мышь и тач),
  выбор детали кликом, перемещение за стрелки-гизмо с привязкой к 1 мм.
- **Деталировка** — таблица с агрегацией одинаковых деталей, итоги по площади (м²)
  и длине кромки (м.п.). Экспорт в **CSV** и в **JSON-проект**, печать.
- **Материалы** — справочник с толщиной и цветом (цвет используется в 3D).
- **Хранение** — автосохранение в IndexedDB (офлайн), импорт/экспорт файла проекта.
- **История** — Undo/Redo (Ctrl/Cmd+Z, Shift для redo).

## Стек

Vite · React · TypeScript · three / @react-three/fiber / @react-three/drei ·
Zustand · idb-keyval · vite-plugin-pwa.

## Запуск

```bash
npm install
npm run dev      # разработка
npm run build    # продакшн-сборка в dist/
npm run preview  # просмотр собранной версии
```

## Модель данных

Все размеры — в миллиметрах. Деталь хранит две большие стороны панели
(`length`, `width`); толщина берётся из материала. `orientation` определяет,
как деталь стоит в корпусе (плашмя / боковина / лицом вперёд).
В модели уже заложено поле `holes` (точки присадок) под **карту присадок** —
это следующая фаза.

## Дорожная карта

- **Фаза 2 — Карта присадок:** расстановка присадок (конфирматы, шканты,
  минификсы, петли, полкодержатели), привязка к соединениям деталей,
  экспорт карты сверловки.
- Раскрой листа (оптимизация карты раскроя), стыковка/прилипание деталей,
  библиотека фурнитуры, облачная синхронизация.
